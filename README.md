#### Way to installation
1. Clone the repo
```
cd /opt && sudo git clone https://github.com/illustratumq/NSv5
```
2. Replace runfile into Desktop:
```
sudo mv NSUv5 /home/$USERNAME/Desktop/
```
3. Run programm
